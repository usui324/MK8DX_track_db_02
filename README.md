# マリオカート 8DX ラウンジ戦績記録 VBA

## 対応バージョン

マリオカート 8DX コース追加パス**第 6 弾**に対応済み。

## 概要

マリオカート 8DX のラウンジ等での戦績を記録し、結果を分析できるツール。  
VBA でデータを記録し、ピボットテーブルでそれを分析する。  
[以前作成したもの](https://github.com/usui324/MK8DX_track_db)の改良版。
